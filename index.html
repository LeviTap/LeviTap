Introduction

Welcome to LeviTap, a program that empowers you to navigate your computer effortlessly using hand gestures as well as use your voice to type! This innovative solution allows you to control your cursor with simple and intuitive movements, enhancing your computing experience. For a comprehensive guide to using the program effectively, click here.

Getting Started

  1. Ensure you have Python installed.
  2. Then download the requirements.txt and run the following command on the command line / terminal.
    $pip3 install -r requirements.txt
  3. Download and install the program linked below.
  4. Follow the on-screen instructions for a seamless setup.

Tips for Precision

  • Maintain a comfortable distance from your camera for accurate gesture recognition.
  • Practice fluid and deliberate hand movements for optimal control.
